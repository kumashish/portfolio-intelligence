# NIFTY 50 Trend Signal Backtest

Dataset: `data/market/nifty50_25years_ohlcv_1999_2026.csv`

Run command:

```bash
uv run pie backtest-market ^NSEI --data-path data/market/nifty50_25years_ohlcv_1999_2026.csv
```

## Result

| Metric | Value |
| --- | ---: |
| Closed signals | 377 |
| Win rate | 39.5% |
| Average directional move | 0.68% |
| Cumulative return | 255.41% |
| Maximum drawdown | -48.41% |

## Assumptions

- The result measures directional NIFTY index returns, not option debit-spread profit and loss.
- Bull and Strong Bull regimes open long exposure; Bear and Strong Bear regimes open short exposure.
- Signals exit on regime reversal, after 37 calendar days, or at the end of available data.
- This is historical research, not a prediction or a trading recommendation.
